Community notes

Community Code Review[mas] - request.js (javascript) Tuesday, December 22 & 7:00 p.m. Los Angeles time. Let's get together virtually and do another community code review. In our community code reviews, we get together to hangout and talk about the craftsmanship of Other Peoples' Code. I'll select a few representative snippets of code from various trending open source projects, share them on Google Hangouts, and they we can discuss. RSVP on Meetup

Software Craftsmanship North America was here in Los Angeles a few months ago (November 6, 2015). Watch for some recordings on 8th Light's Vimeo page here: https://vimeo.com/eighthlight

Join us on Slack at #los-angeles. Get an invite here.
